# ROMEL Mobile Command Waveform

**Summary:** Mobile-first, high-density tactical command node for Romel's 10 AI agents, styled around the glowing steel-blue optic waveform and raw security terminal register.
**Goal:** Establish a real deployed mobile web application optimized for iOS/Android viewports where Romel can authorize keys, select active agents, trigger live voice-synthesizer feedback, and dispatch protocols via a simulation console.
**Brand:** `romel-brand.md`
**Must-haves:**
- Pre-auth security gate (numeric security token input).
- Interactive, animated, glowing steel-blue Waveform canvas reacting to input.
- Roster selector (Grace, Noah, Tom, Felix, Mia, Reeve, Atlas) to switch active agent contexts.
- Monospace rolling diagnostic log stream.
- Web Audio API synthesizer for retro-future key clicks and data sweeps.
- Zero-radius layout, pure `#080C12` deep navy-black background, `#3D8EF0` accents, high density.
- Optimized for mobile viewports (portrait 390px-430px wide, touch-friendly touch targets without rounding).
